## Description

📙 Description des fonctionnalités

Copiez ou exportez rapidement les conversations ChatGPT depuis le bouton de la barre latérale, téléchargez et sauvegardez l'historique de chat en un clic.

📋 Formats d'exportation pris en charge

Exportez les conversations ChatGPT vers Markdown, capture d'écran, PDF, HTML, Excel, etc.
Rendez-vous sur la page d'accueil pour voir tous les formats pris en charge :
https://exportgpt.pro/

Comment mettre à jour vers ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Fonctionnalités prises en charge

- Aperçu, modification, copie, téléchargement des fichiers de format cible
- Paramètres d'exportation : personnalisez les boutons d'exportation à afficher
- Paramètres d'exportation : insérer l'heure de mise à jour du chat dans le nom du fichier
- Paramètres d'exportation : masquer la barre latérale lors de l'initialisation
- Prise en charge de l'exportation du contenu des liens partagés ChatGPT
- Prise en charge du contenu d'image
- Prise en charge du contenu de dialogue d'exportation personnalisé : tout sélectionner, sélection inverse, sélectionner uniquement les réponses

⚒️ Conseils d'utilisation

1. Ouvrez ChatGPT, commencez à discuter avec ChatGPT
   https://chat.openai.com
2. Le plugin ExportGPT affichera une colonne de boutons de barre latérale sur le côté droit de la page, cliquez sur le format que vous souhaitez exporter
3. Dans le panneau contextuel, vous pouvez prévisualiser, modifier, copier ou télécharger le contenu à exporter
