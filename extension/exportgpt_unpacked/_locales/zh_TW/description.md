## Description

📙 功能介紹

從 ChatGPT 側邊欄按鈕快速複製或匯出對話，只需一鍵即可下載並儲存聊天記錄。

📋 支援的匯出格式

將 ChatGPT 對話匯出為 Markdown、截圖、PDF、HTML、Excel 等格式。
前往主頁查看所有支援的格式：
https://exportgpt.pro/

如何升級為 ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ 支援的功能

- 預覽、修改、複製、下載目標格式檔案
- 匯出設定：自訂需要顯示的匯出按鈕
- 匯出設定：檔案名稱中插入聊天更新時間
- 匯出設定：初始化時隱藏側邊欄
- 支援匯出 ChatGPT 分享連結內容
- 支援匯出圖片內容
- 支援自訂匯出對話內容：全選、反選、僅選擇回答內容

⚒️ 使用提示

1. 開啟 ChatGPT，開始與 ChatGPT 對話
   https://chat.openai.com
2. ExportGPT 外掛會在頁面右側顯示一列側邊欄按鈕，點擊你想要匯出的格式
3. 在彈出的面板中，你可以預覽、修改、複製或下載要匯出的內容
