Portuguese (Brazil)

## Description

📙 Descrição das funcionalidades

Copie ou exporte rapidamente conversas do ChatGPT pelo botão da barra lateral, baixe e salve o histórico de chat com um clique.

📋 Formatos de exportação suportados

Exporte conversas do ChatGPT para Markdown, screenshot, PDF, HTML, Excel, etc.
Vá para a página inicial para ver todos os formatos suportados:
https://exportgpt.pro/

Como atualizar para ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Funcionalidades suportadas

- Visualizar, modificar, copiar, baixar arquivos no formato de destino
- Configurações de exportação: personalize os botões de exportação a serem exibidos
- Configurações de exportação: inserir tempo de atualização do chat no nome do arquivo
- Configurações de exportação: ocultar barra lateral durante a inicialização
- Suporte para exportar conteúdo de links compartilhados do ChatGPT
- Suporte para conteúdo de imagem
- Suporte para conteúdo de diálogo de exportação personalizado: selecionar tudo, seleção inversa, selecionar apenas respostas

⚒️ Dicas de uso

1. Abra o ChatGPT, comece a conversar com o ChatGPT
   https://chat.openai.com
2. O plugin ExportGPT exibirá uma coluna de botões da barra lateral no lado direito da página, clique no formato que deseja exportar
3. No painel popup, você pode visualizar, modificar, copiar ou baixar o conteúdo a ser exportado
