## Description

📙 Descrizione delle funzionalità

Copia o esporta rapidamente le conversazioni ChatGPT dal pulsante della barra laterale, scarica e salva la cronologia chat con un clic.

📋 Formati di esportazione supportati

Esporta le conversazioni ChatGPT in Markdown, screenshot, PDF, HTML, Excel, ecc.
Vai alla homepage per vedere tutti i formati supportati:
https://exportgpt.pro/

Come aggiornare a ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Funzionalità supportate

- Anteprima, modifica, copia, download dei file nel formato di destinazione
- Impostazioni esportazione: personalizza i pulsanti di esportazione da visualizzare
- Impostazioni esportazione: inserisci il tempo di aggiornamento della chat nel nome del file
- Impostazioni esportazione: nascondi la barra laterale durante l'inizializzazione
- Supporto per l'esportazione del contenuto dei link condivisi ChatGPT
- Supporto per il contenuto delle immagini
- Supporto per il contenuto del dialogo di esportazione personalizzato: seleziona tutto, selezione inversa, seleziona solo le risposte

⚒️ Suggerimenti per l'uso

1. Apri ChatGPT, inizia a chattare con ChatGPT
   https://chat.openai.com
2. Il plugin ExportGPT mostrerà una colonna di pulsanti della barra laterale sul lato destro della pagina, clicca sul formato che vuoi esportare
3. Nel pannello popup, puoi visualizzare in anteprima, modificare, copiare o scaricare il contenuto da esportare
