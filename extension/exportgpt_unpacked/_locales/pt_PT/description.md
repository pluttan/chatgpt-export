## Language

Portuguese (Portugal)

## Description

📙 Funcionalidade

Copie ou exporte rapidamente conversas do ChatGPT através do botão da barra lateral para descarregar e guardar o histórico de chat com um clique.

📋 Formatos suportados

Exporte conversas do ChatGPT para Markdown, Captura de ecrã, PDF, HTML, Excel, etc.
Visite a página inicial para ver todos os formatos suportados
https://exportgpt.pro/

Como actualizar para ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Funcionalidades suportadas

- Pré-visualizar, modificar, copiar, descarregar ficheiros no formato de destino
- Definições de exportação: Personalizar os botões de exportação que precisam de ser exibidos
- Definições de exportação: Inserir hora de actualização do chat no nome do ficheiro
- Definições de exportação: Ocultar barra lateral durante a inicialização
- Suporte para exportar ligações partilhadas do ChatGPT
- Suporte para conteúdo de imagem
- Suporte para personalizar conteúdo de diálogo de exportação: seleccionar tudo, selecção inversa, seleccionar apenas respostas

⚒️ Dicas de utilização

1. Abra o ChatGPT e inicie uma conversa com o ChatGPT
   https://chat.openai.com
2. O plugin ExportGPT irá colocar uma coluna de botões da barra lateral no lado direito da página, clique no formato que deseja exportar
3. No painel que aparece, pode pré-visualizar, modificar, copiar ou descarregar o conteúdo a ser exportado

🔗 Ligações

- Exporte qualquer conteúdo web para formato Markdown: https://chatopenai.pro/web2markdown/
