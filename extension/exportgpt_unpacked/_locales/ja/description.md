## Description

📙 機能紹介

ChatGPT サイドバーボタンから対話を素早くコピーまたはエクスポートし、ワンクリックでチャット履歴をダウンロード・保存できます。

📋 サポートされるエクスポート形式

ChatGPT 対話を Markdown、スクリーンショット、PDF、HTML、Excel などの形式でエクスポート。
すべてのサポート形式については、ホームページをご覧ください：
https://exportgpt.pro/

ExportGPT Pro へのアップグレード方法
https://exportgpt.pro/pricing

⚙️ サポートされる機能

- ターゲット形式ファイルのプレビュー、編集、コピー、ダウンロード
- エクスポート設定：表示するエクスポートボタンのカスタマイズ
- エクスポート設定：ファイル名にチャット更新時間を挿入
- エクスポート設定：初期化時にサイドバーを非表示
- ChatGPT 共有リンクコンテンツのエクスポートをサポート
- 画像コンテンツをサポート
- カスタムエクスポート対話コンテンツをサポート：全選択、逆選択、回答のみ選択

⚒️ 使用方法

1. ChatGPT を開き、ChatGPT との対話を開始
   https://chat.openai.com
2. ExportGPT プラグインがページ右側にサイドバーボタンを表示するので、エクスポートしたい形式をクリック
3. ポップアップパネルで、エクスポートするコンテンツをプレビュー、編集、コピー、またはダウンロード

🔗 リンク

- 任意の Web コンテンツを Markdown 形式でエクスポート：https://chatopenai.pro/web2markdown/
