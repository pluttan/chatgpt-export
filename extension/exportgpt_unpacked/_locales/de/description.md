## Description

📙 Funktionsbeschreibung

Kopieren oder exportieren Sie ChatGPT-Unterhaltungen schnell über die Seitenleisten-Schaltfläche, laden Sie den Chat-Verlauf mit einem Klick herunter und speichern Sie ihn.

📋 Unterstützte Exportformate

Exportieren Sie ChatGPT-Unterhaltungen nach Markdown, Screenshot, PDF, HTML, Excel usw.
Besuchen Sie die Homepage, um alle unterstützten Formate zu sehen:
https://exportgpt.pro/

So aktualisieren Sie auf ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Unterstützte Funktionen

- Vorschau, Bearbeitung, Kopieren, Download von Zielformat-Dateien
- Exporteinstellungen: Anpassen der anzuzeigenden Export-Schaltflächen
- Exporteinstellungen: Chat-Aktualisierungszeit in Dateinamen einfügen
- Exporteinstellungen: Seitenleiste bei der Initialisierung ausblenden
- Unterstützung für den Export von ChatGPT-geteilten Link-Inhalten
- Unterstützung für Bildinhalte
- Unterstützung für benutzerdefinierten Export-Dialog-Inhalt: Alle auswählen, Auswahl umkehren, nur Antworten auswählen

⚒️ Verwendungstipps

1. Öffnen Sie ChatGPT, beginnen Sie mit ChatGPT zu chatten
   https://chat.openai.com
2. Das ExportGPT-Plugin zeigt eine Spalte von Seitenleisten-Schaltflächen auf der rechten Seite der Seite an, klicken Sie auf das Format, das Sie exportieren möchten
3. Im Popup-Panel können Sie den zu exportierenden Inhalt vorschauen, bearbeiten, kopieren oder herunterladen
