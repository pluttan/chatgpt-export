## Description

📙 Popis funkcí

Rychle kopírujte nebo exportujte konverzace ChatGPT z tlačítka postranního panelu, stáhněte a uložte historii chatu jedním kliknutím.

📋 Podporované formáty exportu

Exportujte konverzace ChatGPT do formátů Markdown, snímek obrazovky, PDF, HTML, Excel atd.
Přejděte na domovskou stránku a zobrazte všechny podporované formáty:
https://exportgpt.pro/

Jak upgradovat na ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Podporované funkce

- Náhled, úprava, kopírování, stahování souborů cílového formátu
- Nastavení exportu: přizpůsobte tlačítka exportu, která se mají zobrazit
- Nastavení exportu: vložte čas aktualizace chatu do názvu souboru
- Nastavení exportu: skryjte postranní panel během inicializace
- Podpora exportu obsahu sdílených odkazů ChatGPT
- Podpora obsahu obrázků
- Podpora vlastního obsahu dialogu exportu: vybrat vše, inverzní výběr, vybrat pouze odpovědi

⚒️ Tipy pro použití

1. Otevřete ChatGPT, začněte chatovat s ChatGPT
   https://chat.openai.com
2. Plugin ExportGPT zobrazí sloupec tlačítek postranního panelu na pravé straně stránky, klikněte na formát, který chcete exportovat
3. V popup panelu můžete náhled, úpravu, kopírování nebo stahování obsahu k exportu
