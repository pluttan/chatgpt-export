## Language

Swedish

## Description

📙 Funktion

Kopiera eller exportera snabbt ChatGPT-konversationer från sidofältsknappen för att ladda ner och spara chatthistorik med ett klick.

📋 Stödda format

Exportera ChatGPT-konversationer till Markdown, Skärmbild, PDF, HTML, Excel, etc.
Gå till startsidan för att se alla stödda format
https://exportgpt.pro/

Hur man uppgraderar till ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Stödda funktioner

- Förhandsgranska, ändra, kopiera, ladda ner filer i målformat
- Exportinställningar: Anpassa exportknappar som behöver visas
- Exportinställningar: Infoga chattuppdateringstid i filnamn
- Exportinställningar: Dölj sidofält under initialisering
- Stöd för export av ChatGPT delade länkar
- Stöd för bildinnehåll
- Stöd för anpassning av exportdialoginnehåll: välj alla, omvänd markering, välj endast svar

⚒️ Användningstips

1. Öppna ChatGPT och starta en konversation med ChatGPT
   https://chat.openai.com
2. ExportGPT-pluginet kommer att placera en kolumn med sidofältsknappar på höger sida av sidan, klicka på formatet du vill exportera
3. I popup-panelen kan du förhandsgranska, ändra, kopiera eller ladda ner innehållet som ska exporteras

🔗 Länkar
