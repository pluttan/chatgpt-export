## Description

📙 Feature Description

Quickly copy or export ChatGPT conversations from the sidebar button, download and save chat history with one click.

📋 Supported Export Formats

Export ChatGPT conversations to Markdown, screenshot, PDF, HTML, Excel, etc.
Go to the homepage to see all supported formats:
https://exportgpt.pro/

How to Upgrade to ExportGPT Pro
https://exportgpt.pro/pricing

⚙️ Supported Features

- Preview, modify, copy, download target format files
- Export settings: customise export buttons to display
- Export settings: insert chat update time in file name
- Export settings: hide sidebar during initialisation
- Support exporting ChatGPT shared link content
- Support image content
- Support custom export dialogue content: select all, inverse selection, select only answers

⚒️ Usage Tips

1. Open ChatGPT, start chatting with ChatGPT
   https://chat.openai.com
2. The ExportGPT plugin will display a column of sidebar buttons on the right side of the page, click the format you want to export
3. In the popup panel, you can preview, modify, copy or download the content to be exported
